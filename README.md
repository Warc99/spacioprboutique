# Spacio PR Boutique

Pequeña web estática de una tienda de joyas hechas a mano.

Instrucciones rápidas

- Preparar y subir a GitHub:
  1. Inicializar repo local: `git init`
  2. Añadir archivos: `git add .`
  3. Commit inicial: `git commit -m "Initial commit"`
  4. Crear repo en GitHub (o usar uno existente) y añadir remote:
     `git remote add origin <URL-REPO>`
  5. Subir: `git push -u origin main`

- Desplegar en Vercel (CLI):
  - Instalar: `npm i -g vercel` (si no está instalado)
  - Desplegar: `vercel --prod` y seguir el asistente

Archivos relevantes
- `index.html`, `styles.css`, `script.js`, `productos.js`, carpeta `imagenes/`, `redes/`

Notas
- Si no deseas usar la CLI, puedes conectar el repositorio en https://vercel.com desde tu cuenta y desplegar directamente desde GitHub.
